# Blender 新手建模块速入门

## —— 面向 SionnaEM 项目的零基础建模指南

## 第0章：教程概述

### 这份教程写给谁？

**完全的 Blender 初学者**——你以前从未打开过 Blender，甚至可能没碰过任何 3D 建模软件。没关系，这份教程从零开始，每一步都给出具体的按键和数值。

### 你将学到什么？

1. 在 Windows 上安装 Blender 并设置为中文界面
2. 认识 Blender 的基本界面和操作方式
3. 从头建模一架四旋翼无人机（quadrotor UAV）
4. 将模型导出为 Sionna RT 可用的 PLY 文件格式
5. 生成 Mitsuba XML 场景文件，并在 Sionna RT 中加载

### 学完后的成果

你将拥有一架尺寸精确的四旋翼飞行器模型（8 个部件），可以导出到 Sionna RT 中进行电磁仿真和微多普勒分析。

```
        俯视图（Z 轴向下看）

             电机3
              |
              |
       臂2 ——+—— 臂1 —— 电机1
              |
         [机身]  
              |
       臂3 ——+—— 臂4
              |
             电机4
              |
             电机2
```

### 本教程不涵盖

- 材质、纹理、贴图（Sionna RT 使用自己的无线电材料系统）
- 动画、骨骼绑定
- 雕刻、复杂曲面建模
- Cycles/Eevee 渲染

### 前提条件

- 一台 Windows 电脑（Blender 对显卡没有硬性要求，集成显卡即可建模）
- 能够访问 [blender.org](https://www.blender.org/download/) 下载软件
- 约 2-3 小时的学习时间

## 第1章：Blender 安装（Windows）

### 1.1 下载 Blender

1. 打开浏览器，访问 **https://www.blender.org/download/**
2. 页面会自动检测你的系统，显示一个大按钮 **"Download Blender 4.2 LTS"**（或更新版本）
3. 点击按钮下载安装程序（约 300 MB）

> **为什么选 LTS 版本？** LTS = 长期支持版，Bug 更少，兼容性更好。4.2 LTS 是目前（2025）推荐的选择。

### 1.2 安装 Blender

1. 双击下载的 `blender-4.2.*-windows-x64.msi` 文件
2. 点击 **Next**，接受许可协议
3. 安装选项中，建议全部勾选：
   - ✅ **Add Blender to PATH**（方便命令行使用）
   - ✅ **Open .blend files with Blender**（关联文件格式）
4. 选择安装路径（默认 `C:\Program Files\Blender Foundation\Blender 4.2\`）
5. 点击 **Install**，等待安装完成

### 1.3 首次启动与中文设置

1. 双击桌面上的 Blender 图标启动
2. 首次启动会出现 **快速设置（Quick Setup）** 对话框——直接点击空白处关闭
3. 进入主界面后，设置中文：
   - 菜单栏：**Edit → Preferences**（编辑 → 偏好设置）
   - 左侧选择 **Interface**（界面）
   - 在 **Translation**（翻译）区域：
     - 将 **Language** 从 `English` 改为 **`Simplified Chinese (简体中文)`**
     - 勾选下面出现的三个复选框：
       - ✅ **Tooltips**（工具提示）
       - ✅ **Interface**（界面）
       - ✅ **New Data**（新建数据）
   - 界面会立即变成中文

> **提示**：本教程中会同时给出中英文菜单名称，方便你对比。如果你习惯英文界面，可以保持英文，不影响后续操作。

### 1.4 确认安装成功

- 你应该能看到一个显示默认立方体的 3D 视图窗口
- 菜单栏显示为中文（如"文件""编辑""渲染"等）
- 如果界面正常，安装完成！可以进入下一章

## 第2章：界面巡览

### 2.1 默认布局

启动 Blender 后，你会看到类似这样的布局：

```
┌──────────────────────────────┬─────────────┐
│                              │             │
│      3D 视口                 │  大纲视图    │
│      (3D Viewport)          │ (Outliner)  │
│                              │             │
│    [这里是默认的立方体]       ├─────────────┤
│                              │             │
│                              │  属性编辑器   │
│                              │ (Properties) │
├──────────────────────────────┴─────────────┤
│          时间线 (Timeline)                  │
└────────────────────────────────────────────┘
```

四个主要区域：

| 区域 | 中文名 | 位置 | 功能 |
|------|--------|------|------|
| 3D Viewport | 3D 视口 | 中央最大区域 | 观察和操作 3D 物体，你大部分时间在这里 |
| Outliner | 大纲视图 | 右上 | 场景中所有物体的列表，按名称排列 |
| Properties | 属性编辑器 | 右下 | 物体的详细属性、修改器、材质等 |
| Timeline | 时间线 | 底部 | 动画时间轴（本教程不使用） |

### 2.2 工作区（Workspace）选项卡

顶部有一排选项卡：**布局 / 建模 / 雕刻 / UV编辑 / 纹理绘制 / 着色 / 动画 / 渲染 / 合成 / 几何节点**

- 点击每个选项卡，窗口布局会切换为对应工作流优化
- 本教程主要使用 **布局** 和 **建模** 两个工作区

> **[图1: 工作区选项卡栏，高亮显示"建模"选项卡]**

### 2.3 工具栏和侧边栏

- **工具栏**：位于 3D 视口左侧，包含常用工具（选择、移动、旋转、缩放等）。按 **T** 键可显示/隐藏
- **侧边栏**：按 **N** 键弹出右侧面板，显示选中物体的精确变换数值（位置、旋转、缩放）

### 2.4 基本概念

| 术语 | 英文 | 含义 |
|------|------|------|
| 物体 | Object | 场景中的一个独立实体，如一个立方体、一个圆柱体 |
| 网格 | Mesh | 由顶点、边、面构成的 3D 形状 |
| 顶点 | Vertex | 空间中的一个点（构成网格的最小单元） |
| 边 | Edge | 连接两个顶点的线段 |
| 面 | Face | 由 3 条或更多边围成的平面（三角形/四边形/n边形） |
| 三角面 | Triangle | 只有 3 个顶点的面——Sionna RT 要求所有面必须是三角面 |

## 第3章：基础导航与操作

### 3.1 视口控制（必须在 3D 视口中操作）

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| **旋转视角** | **鼠标中键 拖动** | 围绕视口中心旋转摄像机 |
| **平移视角** | **Shift + 鼠标中键 拖动** | 上下左右平移摄像机 |
| **缩放** | **滚轮滚动** | 拉近/拉远视角 |
| 顶视图 | 数字键盘 **7** | 从正上方看（俯视） |
| 前视图 | 数字键盘 **1** | 从正前方看 |
| 右视图 | 数字键盘 **3** | 从正右方看 |
| 聚焦选中物体 | 数字键盘 **.** (句号) | 将视角拉到选中物体近处 |

> **没有数字键盘？** 在 Edit → Preferences → 输入 → 键盘，勾选 **"模拟数字键盘"**，之后可以用主键盘顶部的数字键代替。

> **[图2: 不同视图角度下的默认立方体——顶视图、前视图、右视图对比]**

### 3.2 选择操作

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 选择物体 | **鼠标左键 点击** | 单击选中一个物体（橙色轮廓） |
| 框选 | **B** 键，然后拖动 | 画一个矩形框，框内的物体全选 |
| 加选 | **Shift + 左键点击** | 不取消已选物体的同时添加选择 |
| 减选 | **Shift + 左键点击已选物体** | 取消某个物体的选中状态 |
| 全选 | **A** 键 | 按一次全选，再按一次取消全选 |
| 取消所有选择 | **Alt + A** 或 **双击 A** | 清空选择 |

### 3.3 变换操作（移动 / 旋转 / 缩放）

这是最核心的操作，**务必记住这三个快捷键**：

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| **移动**（抓取） | **G** (Grab) | 自由移动选中物体。移动鼠标确认位置，**左键确定** / **右键取消** |
| **旋转** | **R** (Rotate) | 自由旋转选中物体 |
| **缩放** | **S** (Scale) | 自由缩放选中物体 |

**轴向锁定**（非常重要）：

按 G/R/S 后，再按 X / Y / Z 可将操作限制在对应轴上：
- `G` → `X` → 输入数字 → 沿 X 轴移动指定距离
- `R` → `Z` → 输入 `45` → 绕 Z 轴旋转 45 度
- `S` → `Y` → 输入 `2` → 沿 Y 轴缩放为 2 倍

**精确输入**：按 G/R/S 后直接输入数字（如 `G X 0.15` 表示沿 X 轴移动 0.15 米），按 Enter 确认。

> **[图3: 选中立方体后按 G 键移动，显示移动轴线的场景]**

### 3.4 物体模式 vs 编辑模式

| 模式 | 切换方式 | 用途 |
|------|---------|------|
| **物体模式** (Object Mode) | 默认模式 | 移动/旋转/缩放**整个物体**，添加新物体 |
| **编辑模式** (Edit Mode) | 选中物体后按 **Tab** 键 | 修改物体的**顶点/边/面** |

按 **Tab** 键在两个模式间切换。这是 Blender 最常用的切换键之一。

在编辑模式下，可以按 **1/2/3** 键切换选择模式：
- **1** = 顶点选择
- **2** = 边选择
- **3** = 面选择

### 3.5 添加和删除物体

| 操作 | 方法 |
|------|------|
| **添加物体** | **Shift + A** → 选择"网格" → 选择形状（立方体、圆柱体、平面等） |
| **删除物体** | 选中物体 → **X** 键或 **Delete** 键 → 确认删除 |

### 3.6 复制物体

| 操作 | 快捷键 |
|------|--------|
| **复制** | **Shift + D**（复制后立即进入移动模式，移动鼠标放置，左键确认） |
| **复制 + 保持位置** | Shift + D → 右键（取消移动，新物体留在原位） |

### 3.7 撤销与重做

| 操作 | 快捷键 |
|------|--------|
| 撤销 | **Ctrl + Z** |
| 重做 | **Ctrl + Shift + Z** |

### 3.8 重命名物体

双击大纲视图(Outliner)中的物体名称，输入新名称后按 Enter。
或者选中物体后按 **F2** 键，输入名称。

> **命名很重要**！Sionna RT 根据 PLY 文件名来分配材料，所以请使用英文命名（如 `drone_body`、`drone_arm_1`）。本教程中会提示每个物体的命名。

## 第4章：四旋翼飞行器分步建模

这是教程的核心部分。你将从头创建一架完整的四旋翼无人机。

### 4.1 准备工作：单位、坐标与场景清理

#### 设置场景单位为"米"

Sionna RT 使用米作为单位，Blender 默认也是 1 单位 = 1 米，我们确认一下：

1. 在右侧 **属性编辑器** 中，点击左侧第二个图标（场景属性，图标是锥形+圆柱+球）
2. 展开 **单位（Units）** 面板
3. 确认 **单位系统（Unit System）** 为 **无（None）**（Blender 默认无单位系统时，1 单位 = 1 米）
4. 或者在 **长度（Length）** 中选择 **公制（Metric）**，并将单位缩放设为 `1.000000`

> **[图4: 属性编辑器中场景单位的设置位置]**

#### 坐标系确认

Blender 默认坐标系：
- **X 轴**：左右（红色）
- **Y 轴**：前后（绿色）
- **Z 轴**：上下（蓝色）

这与 Sionna RT 要求的 **Y Forward, Z Up** 一致，无需修改。

#### 删除默认立方体

1. 用鼠标左键点击场景中的默认立方体（选中后呈橙色轮廓）
2. 按 **X** 键
3. 在弹出的菜单中点击 **删除（Delete）**

场景现在是空的，准备开始建模。

### 4.2 创建机身

机身是一个长方体：**宽 0.30 × 深 0.30 × 高 0.06 米**。

#### 步骤1：添加立方体

1. 按 **Shift + A**
2. 选择 **网格（Mesh）→ 立方体（Cube）**
3. 一个 2×2×2 米的立方体出现在原点 (0,0,0)

#### 步骤2：设置精确尺寸

1. 确保在 **物体模式**（如果不是，按 Tab 退出编辑模式）
2. 在 3D 视口选中刚添加的立方体
3. 按 **N** 键打开右侧边栏
4. 在 **变换（Transform）** 面板的 **尺寸（Dimensions）** 部分：
   - X（宽）：输入 `0.30`
   - Y（深）：输入 `0.30`
   - Z（高）：输入 `0.06`
5. 在 **位置（Location）** 部分，确认 X/Y/Z 都是 `0.000`（机身居中）

> **[图5: 右侧 N 面板中设置机身尺寸的截图]**

#### 步骤3：重命名

1. 在大纲视图（右上角 Outliner）中找到 **"立方体"（Cube）**
2. 双击名称 → 输入 `drone_body` → 按 Enter

> **检查点1**：你应该在 3D 视口中央看到一个扁平的扁盒子，宽 0.30m × 深 0.30m × 高 0.06m。这就是无人机的机身。

### 4.3 创建第一根机臂

机臂是将机身连接到电机的圆柱体：
- **半径**：0.015 m
- **长度**：0.50 m（从机身中心到电机位置的距离 = 0.15 + 0.35）
- **方向**：沿 X 轴水平延伸

#### 步骤1：添加圆柱体

1. 按 **Shift + A** → **网格（Mesh）→ 圆柱体（Cylinder）**
2. 注意：圆柱体默认沿 Z 轴竖立，我们需要把它放平

#### 步骤2：将圆柱体旋转到水平（沿 X 轴）

1. 选中圆柱体，按 **R** 键（旋转）
2. 按 **Y** 键（锁定 Y 轴旋转）
3. 输入 `90` → 按 Enter
   - 现在圆柱体沿 X 轴水平放置

> 如果你更习惯纯数值方式：按 N 打开侧边栏，在 **旋转（Rotation）** 中将 Y 设为 `90°`，X 和 Z 保持 `0°`。

#### 步骤3：设置尺寸

在 N 面板的 **尺寸（Dimensions）** 部分：
- X（长度方向）：输入 `0.50`
- Y（半径方向）：输入 `0.03`（直径 = 半径 0.015 × 2）
- Z（半径方向）：输入 `0.03`

> Blender 中`尺寸`反映的是整个物体的包围盒，此处不做精确径向设置；更精确的方式是使用编辑模式。对于本教程的仿真用途，近似尺寸即可，后续导出时会三角化。

#### 步骤4：定位机臂

在 N 面板的 **位置（Location）** 部分：
- X：`0.25`（从原点向右偏移半个机身 + 半个臂长 = 0 + 0.25）
- Y：`0.000`
- Z：`0.000`

#### 步骤5：重命名

在大纲视图中双击圆柱体名称 → 输入 `drone_arm_1`

> **[图6: 放置好 drone_arm_1 后的顶视图——可以看到扁盒向右伸出一根圆柱体]**

> **检查点2**：切换到顶视图（数字键盘 7），你应该看到机身盒子右边伸出一根沿 X 轴方向的圆柱体，这是机臂1。

### 4.4 创建其余三根机臂

现在你有了一根沿 +X 方向的臂。我们需要向另外三个方向也各放一根。

**方案一：逐个复制旋转（适合新手理解）**

#### 臂 2（-X 方向）：

1. 选中 `drone_arm_1`，按 **Shift + D** 复制
2. 立即按 **X** 键锁定 X 轴，拖动到 `-0.25` 位置
3. 重命名为 `drone_arm_2`
4. 在 N 面板确认位置：X = `-0.25`, Y = `0`, Z = `0`

#### 臂 3（+Y 方向）：

1. 选中 `drone_arm_1`，按 **Shift + D** 复制
2. 按 **R** → **Z** → 输入 `90` → 圆柱体现在沿 Y 轴
3. 在 N 面板设置位置：X = `0`, Y = `0.25`, Z = `0`
4. 重命名为 `drone_arm_3`

#### 臂 4（-Y 方向）：

1. 选中 `drone_arm_3`，按 **Shift + D** 复制
2. 在 N 面板设置位置：X = `0`, Y = `-0.25`, Z = `0`
3. 重命名为 `drone_arm_4`

> **检查点3**：切换到顶视图（数字键盘 7），你应该看到机身向四个方向各伸出一根臂，呈十字形。

### 4.5 创建四个电机

电机是放在每根臂末端的短圆柱体：
- **半径**：0.03 m
- **高度**：0.03 m

#### 电机1（+X 方向臂末端）

1. 按 **Shift + A** → **网格 → 圆柱体**
2. 新圆柱体默认沿 Z 轴竖立（正是我们想要的——电机竖立在臂上方）
3. 在 N 面板设置：
   - **尺寸**：X=0.06, Y=0.06, Z=0.03
   - **位置**：X=`0.50`, Y=`0`, Z=`0.015`
     - X=0.50 是臂的末端（机身中心 0 + 臂长 0.50）
     - Z=0.015 让电机底部与臂表面接触
4. 重命名为 `drone_motor_1`

#### 电机2（-X 方向臂末端）

1. 选中 `drone_motor_1`，按 **Shift + D**
2. 在 N 面板设置位置：X=`-0.50`, Y=`0`, Z=`0.015`
3. 重命名为 `drone_motor_2`

#### 电机3（+Y 方向臂末端）

1. 选中 `drone_motor_1`，按 **Shift + D**
2. 在 N 面板设置位置：X=`0`, Y=`0.50`, Z=`0.015`
3. 重命名为 `drone_motor_3`

#### 电机4（-Y 方向臂末端）

1. 选中 `drone_motor_1`，按 **Shift + D**
2. 在 N 面板设置位置：X=`0`, Y=`-0.50`, Z=`0.015`
3. 重命名为 `drone_motor_4`

> **[图7: 顶视图中完整的四旋翼飞行器——机身+4根臂+4个电机]**

> **检查点4（最终检查）**：在大纲视图（Outliner）中，你应该看到 9 个物体：`drone_body`、`drone_arm_1`~`drone_arm_4`、`drone_motor_1`~`drone_motor_4`。

### 4.6 保存你的工作

1. **Ctrl + S** 或 菜单 **文件 → 保存**
2. 选择保存位置，文件名例如 `my_quadrotor.blend`
3. 点击 **保存为（Save As）**

> **建议**：养成经常按 Ctrl+S 的习惯，防止意外丢失进度。

## 第5章：修改器

修改器是 Blender 的强大工具，可以在不破坏原始几何体的前提下改变网格。本章介绍两个对 Sionna RT 导出至关重要的修改器。

### 5.1 Triangulate（三角化）修改器——必须添加

**为什么需要？** Sionna RT 只支持**三角形面片**。Blender 默认的立方体、圆柱体都包含四边形面（quad），在导出前必须全部转为三角形。

#### 为每个物体添加三角化修改器

逐个操作为每个飞行器部件添加：

1. 在 3D 视口中选中 `drone_body`
2. 点击右侧属性编辑器中的蓝色扳手图标（修改器，Modifiers）
3. 点击 **添加修改器（Add Modifier）**
4. 在弹出的列表中搜索或找到 **Triangulate（三角化）**
5. 点击添加

这就是全部操作。Triangulate 修改器没有需要调整的参数。

6. 依次选中 `drone_arm_1` ~ `drone_arm_4`、`drone_motor_1` ~ `drone_motor_4`，重复上述步骤

> **[图8: 属性编辑器中已添加 Triangulate 修改器的修改器堆栈]**

> **批量操作技巧**：你也可以选中所有部件（按住 Shift 逐个点击），然后一次性全部添加修改器。但逐个添加更清晰，不容易遗漏。

### 5.2 Mirror（镜像）修改器——省力技巧（选读）

如果你只需要建四分之一的飞行器，可以用 Mirror 修改器自动生成对称部分。对于已经全建好的模型，可以跳过本节。

Mirror 的基本用法：
1. 选中需要镜像的物体
2. 添加 **Mirror（镜像）** 修改器
3. 选择镜像轴（X / Y / Z）
4. 完成——对原物体的任何编辑都会自动镜像到对面

> 对于四旋翼飞行器这种简单几何体，手动复制反而更快。Mirror 修改器在建模复杂对称物体（如人脸、车辆）时更有用。

## 第6章：导出与 Sionna RT 集成

模型建好后，下一步是将它导出为 Sionna RT 可以加载的格式。

### 6.1 SionnaEM 场景工作流概览

```
Blender 建模 → 导出 PLY 文件 → 创建 materials.json → 运行 generate_scene_xml.py → 生成 .xml → Sionna RT 加载
```

你的目录最终结构应该是：

```
my_drone_scene/
├── meshes/                    ← 存放所有 .ply 文件
│   ├── drone_body.ply
│   ├── drone_arm_1.ply
│   ├── drone_arm_2.ply
│   ├── drone_arm_3.ply
│   ├── drone_arm_4.ply
│   ├── drone_motor_1.ply
│   ├── drone_motor_2.ply
│   ├── drone_motor_3.ply
│   ├── drone_motor_4.ply
│   └── ground.ply             ← 如需地面
├── materials.json             ← 你手动创建
└── my_drone_scene.xml         ← 自动生成
```

### 6.2 导出 PLY 文件

#### 一次导出一个物体

1. 在 3D 视口中只选中 `drone_body`（确保没有其他物体被选中）
2. 菜单：**文件（File）→ 导出（Export）→ Stanford PLY (.ply)**
3. 右侧会出现导出选项面板：

| 设置项 | 中文名 | 必须设置的值 |
|--------|--------|-------------|
| Format | 格式 | **ASCII** |
| Forward Axis | 向前轴 | **Y Forward** |
| Up Axis | 向上轴 | **Z Up** |
| Scale | 缩放 | **1.0** |
| Selected Only | 仅选中物体 | ✅ 勾选 |
| Apply Modifiers | 应用修改器 | ✅ 勾选 |
| Triangulate | 三角化 | ✅ 勾选 |
| UV Coordinates | UV 坐标 | 不勾选（可选） |
| Color Attributes | 顶点颜色 | 不勾选 |

> **[图9: PLY 导出对话框，各设置项已按要求配置]**

4. 选择保存位置。建议先在桌面上创建 `my_drone_scene` 文件夹，内部创建 `meshes` 子文件夹
5. 文件名输入 `drone_body.ply`，保存到 `meshes/` 文件夹
6. 点击 **导出 PLY（Export PLY）**

#### 逐个导出全部 9 个部件

按相同设置，依次导出每个部件到 `meshes/` 文件夹：
- `drone_body.ply`
- `drone_arm_1.ply` ~ `drone_arm_4.ply`
- `drone_motor_1.ply` ~ `drone_motor_4.ply`

> **也可以导出地面**：在 Blender 中添加一个平面（Shift+A → 网格 → 平面），缩放到合适大小（如 10m×10m），放在 Z=0，导出为 `ground.ply`。

### 6.6 在 Sionna RT 中加载

启动你的 Sionna RT Python 环境，运行以下代码：

```python
import sionna
from sionna.rt import load_scene, PlanarArray, Transmitter, Receiver, Camera

# 加载你建模的场景
scene = load_scene("my_drone_scene/my_drone_scene.xml")
scene.frequency = 3.5e9  # 工作频率 3.5 GHz

# 配置天线
scene.tx_array = PlanarArray(num_rows=1, num_cols=1, pattern="iso", polarization="V")
scene.rx_array = PlanarArray(num_rows=1, num_cols=1, pattern="iso", polarization="V")

# 放置发射机（基站）
tx = Transmitter(name="bs_0", position=[5.0, 0.0, 8.0], power_dbm=44)
scene.add(tx)

# 将接收机放在无人机附近
rx = Receiver(name="ue", position=[0.0, 0.0, 1.0])
scene.add(rx)

# 预览场景（Jupyter 中可以看到 3D 渲染）
scene.preview()

# 计算传播路径
from sionna.rt import PathSolver
paths = PathSolver()(scene, max_depth=5)
print(f"找到 {paths.a.shape[-1]} 条路径")
```

如果一切正常，`scene.preview()` 会显示你建的无人机模型。

> **如果场景加载失败**，请检查：
> 1. `meshes/` 文件夹与 `.xml` 文件是否在同一目录下
> 2. PLY 文件是否全部是三角面片（确认导出了 Triangulate 选项）
> 3. `materials.json` 中的名称是否与 `.ply` 文件名（不含扩展名）完全一致

## 附录C：常见问题排查

### Q1：PLY 文件无法在 Sionna RT 中显示

- 确认 PLY 导出时勾选了 **三角化（Triangulate）**
- 确认每个物体都添加了 **Triangulate 修改器**且导出时勾选了 **应用修改器**
- 检查 PLY 格式：用文本编辑器打开 `.ply` 文件，确认第一行是 `ply`，第二行是 `format ascii 1.0`

### Q2：场景加载报错 "Need triangle mesh"

Sionna RT 要求所有面必须是三角形。在 Blender 编辑模式中按 **3** 键（面选择），如果不全是三角形，说明 Triangulate 没有正确应用。回到第 5.1 节检查。

### Q3：材质不对 / 物体全是同一种材料

- 检查 `materials.json` 中的键名是否与 PLY 文件名（不含 `.ply` 扩展名）**完全一致**
- JSON 格式是否合法（可以用 [jsonlint.com](https://jsonlint.com) 验证）
- 未在 `materials.json` 中列出的网格会使用默认材料 `metal`

### Q4：Blender 导出 PLY 时多个物体合并成一个文件

是的，Blender 默认行为。确保每次只选中一个物体，并勾选 **仅选中物体（Selected Only）**。

### Q5：找不到数字键盘

在 **编辑 → 偏好设置 → 输入 → 键盘** 中，勾选 **模拟数字键盘（Emulate Numpad）**。之后主键盘数字键 1-9 可以代替数字键盘。

### Q6：Blender 突然关闭了

Blender 默认有自动保存功能。重新打开 Blender 后，菜单 **文件 → 恢复 → 自动保存**，可以找回最近的文件。
