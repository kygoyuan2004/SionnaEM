一、问题背景与核心问题

总目标：探究 Micro-Doppler 能否集成到 Sionna RT 中，用于仿真多旋翼 UAV 的微多普勒特征。

三个核心问题：

  2. 代码层面的定位 — 这个区别在 Sionna 代码中对应哪个位置？改动点在哪里？
  3. 可行性判断 — 是否只是公式的简单区别，比较好找到位置直接加？

二、公式层面：标准 Doppler vs Micro-Doppler

标准 Doppler

频移是常数，相位为线性斜坡：

a_i(t) = a_i · e^{-j2πfτ_i} · e^{j2π f_Δ t}

其中 f_Δ 由 TX/RX 相对速度决定，每条路径为固定常数。

→ 相位变化：线性，斜率为 2πf_Δ

Micro-Doppler

频移是时变的（旋转叶片产生周期性调制）：

φ(t) = 2π f_Δ t + β·sin(2π f_rot t + φ₀)

调制指数 β = (4πR/λ)·cos(θ)，由叶片半径 R、转速 f_rot、波长 λ 决定。

→ 相位变化：线性斜坡 + 正弦调制

关键差异：标准 Doppler → 频谱上单一频移峰；Micro-Doppler → 频谱上出现以 f_rot 为间距的边带族，形成特征性 "helicopter signature" 图案。

核心结论：Micro-Doppler 可以不修改射线追踪引擎，仅在 CIR 生成阶段给每条路径系数叠加正弦相位调制。射线追踪负责静态场景路径发现，旋转叶片的时变效应通过后处理叠加到路径系数上。

四、三步仿真验证计划

第一步：Baseline — 标准 Doppler 验证 进行中

目标：用 Sionna RT 仿真移动 UE，验证能正确观察到标准 Doppler 频移，建立对照基线。

方法：UE 沿 x 轴匀速移动 (v=10 m/s)，以 Δt=0.5ms 采集 100 个 CIR 快照，从相位差提取 Doppler 频移。

预期：直达径 Doppler ≈ 933 Hz (28 GHz 载频)，相位随时间线性变化。

参数值参数值
载频 fc28 GHzUE 速度10 m/s (沿 x 轴)
采样间隔 Δt0.5 ms快照数100
场景Paris (etoile)

第二步：核心验证 — 单旋转散射点 Micro-Doppler 计划中

目标：证明 Sionna RT 能捕获旋转运动产生的正弦相位调制。

方法：建模单个旋转叶片散射点 (R=0.15m, f_rot=100Hz)，以 2kHz 采样率采集 100 个快照，提取瞬时相位和瞬时频率。

预期：调制指数 β ≈ 175.9 rad，瞬时频率呈正弦波动，相位呈现 φ(t) = 2πf_Δt + βsin(2πf_rot t) 形态。

第三步：应用验证 — 多旋翼 UAV 谱图 计划中

目标：生成完整四旋翼 UAV 的 Micro-Doppler 时频谱图，证明可用于检测/分类。

方法：4 旋翼 / 8 叶片完整 UAV 模型 → CIR 序列 → STFT → Micro-Doppler 谱图。

预期：谱图中清晰可见周期性调制，边带间距 = 100 Hz，呈现经典 helicopter signature。
